# 🌍 Kunpeng Kickoff

> 面向 15-25 岁留学生的「云端行前办事处」—— AI 行程规划 + 众包硬标数据 + 社区软标匹配，三重机制填补出国前的信息断层与决策焦虑。

## 📖 项目简介

Kickoff（出发）致力于解决留学生出国前的核心痛点：信息分散、标准缺失、经验孤岛。通过 **AI 行程规划** 自动生成行前清单、**众包硬标数据库** 量化目的地生活标准、**社区软标匹配** 连接同路人经验，让每一次留学出征都有备而来。

### 三重机制

| 机制 | 解决什么问题 |
|------|-------------|
| 🧠 **AI 行程规划** | "我该准备什么？什么时候办？"——智能生成个性化行前清单与时间线 |
| 📊 **众包硬标数据** | "住哪里安全？吃什么靠谱？"——住宿/餐饮/交通/医疗四大维度量化标准 |
| 👥 **社区软标匹配** | "有没有同校同城的？当地人怎么说？"——搭子匹配 + 问答社区 + 标签云 |

## ✨ 核心功能

| 模块 | 说明 |
|------|------|
| 🧠 **AI 行程规划** | 智能生成行前清单：签证、航班、住宿、入境，按优先级排列 |
| 📊 **硬标数据库** | 目的地住宿、餐饮、交通、医疗硬性标准化数据 |
| 🛡️ **避雷雷达** | 诈骗识别、危险区域预警、商户欺诈提醒 |
| 👥 **社区层** | 同校/同城搭子匹配、Q&A 问答、用户经验标签云 |
| 💰 **商业模式** | 多方收入来源分析与价值主张 |
| 🗺️ **运营路线图** | 分阶段里程碑规划与进度跟踪 |

## 🛠 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite 7
- **样式方案**: Tailwind CSS 3 + shadcn/ui
- **路由**: React Router 7
- **动画**: Framer Motion
- **图表**: Recharts
- **表单**: React Hook Form + Zod
- **主题**: next-themes（深色/浅色模式）

## 🚀 快速开始

### 环境要求

- Node.js >= 18
- npm >= 9

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/liangzheng3017/Kunpeng_Kickoff.git
cd Kunpeng_Kickoff

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

浏览器访问 `http://localhost:5173` 即可查看。

### 其他命令

```bash
npm run build    # 构建生产版本
npm run preview  # 预览生产构建
npm run lint     # 代码检查
```

```
├── types/             # TypeScript 类型定义
├── App.tsx            # 根组件 & 路由配置
└── main.tsx           # 应用入口
```
